# SentinelAI

SentinelAI is a real-time intrusion detection system that monitors network traffic, detects suspicious activity, and visualizes attack patterns through a security dashboard.

## Features
- Real-time packet monitoring
- Suspicious traffic detection
- Attack logging system
- Interactive dashboard visualization

## Technologies
Python
Scapy
Flask
SQLite
Chart.js

## Architecture

Network Traffic
        ↓
Packet Capture (Scapy)
        ↓
Traffic Analysis Engine
        ↓
Attack Detection
        ↓
Database Logging
        ↓
Security Dashboard
